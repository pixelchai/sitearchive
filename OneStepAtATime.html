Like I mentioned in a previous post, I'm actually going to do this project sireously now. First of all, let's start by taking things one step at a time. As it stands, I'm probably not going to be able to do a wide a range of coding languages that I intended to.

Let's start with one basic program, such as a Hello World program for all of the coding languages I wan't to try first, and then we'll expand from there.

Most programmers will make a Hello World application. I'm not "most programmers". So let's come up with a "first program" of our own:

The Higher or Lower Game

This is going to be my "first program". It has the same real-world use as a Hello World program, but it's just slightly more interesting.

You've probably heard of or played this game before, but just in case, here's how it works:

A random number is generated. User tries to guess the number. Computer tells user if they were too high or too low. Keep guessing untill they get the number...

Pseudocode

Here's a basic implementation of it in pseudocode: 


set num = Rand.Next(0, 1000);
set done = false;
do
{
input = GetInput().ToInteger();
    if(input Equals num)
    {
    set done = true
    print("Cngrtz")
    break
    }
    if input greater than num
    {
    print("Too high")
    }
    if input less than num
    {
    print("Too Low")
    }



}
while(not done);
